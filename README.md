# Tetris

## 1. 프로젝트 구성안내
image 파일에 tetris에 사용된 이미지들이 저장되어 있습니다.

music 파일에 tetris에 사용된 음악이 저장되어 있습니다.

gsTetris.xml 파일에 게임 점수기록이 저장되어집니다.

src/org/...파일에 소스코드들이 저장되어 있습니다.
